快速开始

5 分钟完成部署

选择最适合你的安装方式，安装后访问 http://localhost:19377 进入管理面板。

pip 直接运行
Setup Wizard
Docker
# 安装
pip install bridge-server
 
# 设置 API Key（至少一个 Provider）
export DASHSCOPE_API_KEY=sk-xxx
 
# 启动
python -m uvicorn bridge_server.runtime:app --app-dir src --port 19377
 
# 测试第一个请求
curl -X POST http://127.0.0.1:19377/v1/chat/completions \
-H "Authorization: Bearer <token>" \
-d '{"model":"smart","messages":[{"role":"user","content":"你好"}]}'
# 克隆仓库
git clone https://github.com/qiannj/bridge-server.git
cd bridge-server && pip install -e .
 
# 启动交互式 Setup Wizard（引导配置 Provider、路由、认证）
python -m bridge_server.cli.setup_wizard
 
# Wizard 完成后一键启动
python -m bridge_server.cli.setup_wizard --start
# 克隆并配置
git clone https://github.com/qiannj/bridge-server.git
cd bridge-server
cp config.yaml.example config.yaml # 填写 API Key
 
# 启动（含 Nginx 反向代理）
docker-compose up -d
 
# 查看日志
docker-compose logs -f bridge-server
💡 首次启动自动生成 Admin Token 并打印到控制台，管理面板位于 http://localhost:19377
